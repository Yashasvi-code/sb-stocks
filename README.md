SB Stocks
A stock trading simulation app I built to practice full-stack development.

#Features
User authentication
Virtual stock trading
Portfolio tracking
Watchlist management

#Tech Stack
Next.js, TypeScript, Tailwind CSS
Prisma, SQLite
NextAuth.js

#How to Run
npm install
npx prisma db push
npm run dev

#What I Learned
Building REST APIs with Next.js
User authentication with NextAuth
State management with Zustand
Database design with Prisma
